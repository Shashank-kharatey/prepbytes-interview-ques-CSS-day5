Q.1 what are css combinators?

Ans.1 'CSS combinators' are the special type selectors working on the concept of relationship values betweeen the elements.

There are four types of css combinators.

1. 'space' (Descendent selector (all generations/at any position within the body of the parent element))

2. '>' (child selector (only direct child elements))

3. '+' (adjacent sibling (only the imidiate next sibling element))

4. '~' (general sibling (all the next siblings elements

Q.2 what are pseudo selectors?

Ans.2 'pseudo selectors' are the selectors used in some special cases, such as it works when the user pretends or about to work with the selected element.

There are two ways to use pseudo selectors:-

'pseudo class selectors' :-

1. 'Hover' syntax(element name:hover {css property}), it applies the property only after hovering on the element/link
2. 'Active' syntax(element name:active {css property}), it applies the property only during the active stage of the element/link
3. 'Visited' syntax(element name:visited {css property}), it applies the property after the user has selected/activated the element/link
4. 'link' syntax(element name:link {css property}), it applies the properties in the unvisited stage of the element/link

'pseudo element selector' these selectors are used in rare cases, and are used to target the specified part/position of the element. syntax(element name::(pseudo element selector such as,first line,first letter,before,after,marker,and selection) {css property})
